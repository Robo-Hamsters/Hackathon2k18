Garbage collector project.

GitHub

Our application is responsible for finding the shortest route for waste collector vehicles, when picking-up and transporting rubbish to the central waste management location.

You can load given data from a file with pressing the Open File button. Load Markers button will draw and connect all the markers located on the map. Run Algorithm button will find and draw the best route based on our algorithm. Finally, Clear Map button will delete all the markers with their connections from the map.
